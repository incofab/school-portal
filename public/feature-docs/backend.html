Current Feature Notes

  • Manager academic session administration
    Manager-admin routes now expose Route::resource('academic-sessions', Web\AcademicSessions\AcademicSessionController::class)->except(['show']) in routes/manager.php. The controller is app/Http/Controllers/Managers/AcademicSessions/AcademicSessionController.php and uses CreateOrUpdateAcademicSessionRequest with AcademicSession.
  • Class result locking
    Institution routes include POST /class-result-info/{classResultInfo}/lock handled by ClassResultInfoController::updateLock. Result calculation and recalculation lock class result info records; result recording, deletion, and bulk updates use ClassResultInfo::ensureResultIsUnlocked through result actions/controllers before mutating result data.
  • Theory question CRUD
    routes/ccd.php exposes institution-scoped course-session theory question CRUD handled by app/Http/Controllers/CCD/TheoryQuestionController.php and validated by app/Http/Requests/TheoryQuestionRequest.php.
